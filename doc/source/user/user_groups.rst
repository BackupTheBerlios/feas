.. _Benutzergruppen:

===============
Benutzergruppen
===============

Da nicht jedes Mitglied alle Rechte in FEAS haben soll, wurde eine Rechteverwaltung eingebaut. Hier können Benutzergruppen angelegt werden, die diverse Rechte inne haben. Da es bei vielen Anwendern (wie eben in einem Aeroclub) mühselig wäre, jedem Benutzer alle Rechte einzeln zu verleihen, fasst FEAS die vergebenen Rechte zu Benutzergruppen zusammen. In der Mitgliederverwaltung muss dann einfach nur noch eingegeben werden, zu welcher Gruppe eine Person gehört.
