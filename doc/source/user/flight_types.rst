=========
Flugarten
=========

Für die Gebührenabrechnung ist es meistens notwendig, einige Gebührenarten anzulegen. So sollen z.B. oft Schulflüge anders als normale Streckenflüge abgerechnet werden. Andere Flüge zahlt der Verein selbst, wie etwa Werkstattflüge. Hier können die Flugarten angelegt werden und es kann ausgewählt werden, auf wen FEAS den Betrag später buchen soll.
