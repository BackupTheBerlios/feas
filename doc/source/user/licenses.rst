.. _Lizenzen:

========
Lizenzen
========

Jeder Pilot benötigt mindestens eine gültige Fluglizenz und ein medizinisches Tauglichkeitszeugnis. Das Lizenzmodul erleichtert die Überwachung der Mitglieder auf den Besitz der notwendigen Lizenzen. Hier wird festgelegt, welche Lizenzen existieren und wie sich ihre Gültigkeit verhält.
